Features
  • Window-style block of content
  • Realtime caption change
  • Realtime icon on caption change
  • Content lock support
  • Collapse/expand support
  • Collapse/expand event callbacks
  • Drag and Drop support
Getting started
Insert in head
<!-- basic style -->
<link rel="stylesheet" type="text/css" href="your_path/css/mbox.css" />
<!-- required javascripts -->
<script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.9.1/jquery.min.js"></script>
<script type="text/javascript" src="your_path/js/mbox.js"></script>
Insert in body
<div id="my-window" style="width:400px;"> window content </div>

<script type="text/javascript">
$(document).ready( function(){
	$('#my-window').mBox({
		icon		: './img/icon.png',
		caption		: 'My window',
		lockText	: 'Please wait..',
		collapsable	: true,
		collapsed	: false,
		draggable	: true,
		onExpand	: function() { },
		onCollapse	: function() { }
	});
});
</script>
Documentation
Options
iconType: string, Default: null Path to icon image in title of window. captionType: string, Default: " " Text in caption of header. lockTextType: string, Default: "Please wait.." Information message on lock. collapsableType: boolean, Default: false True if collapse/expand features are enabled. collapsedType: boolean, Default: false True if window is in collapsed state on init draggableType: boolean, Default: false True if dragAndDrop features are enabled.
Methods
object.mBox('caption', string)Arguments: text of caption Set new caption in window header object.mBox('icon', string)Arguments: path to image Set new icon in window header object.mBox('expand')Arguments: none Expand window is it's collapsed object.mBox('collapse')Arguments: none Collapse window is it's expanded object.mBox('lock', string)Argument: text message Lock content of window with defined text message. If text message is null, then will be used "lockText" value object.mBox('unlock')Argument: none Unlock content of window
Events
onExpand() Triggered in mBox when window goes to expanded state onCollapse() Triggered in mBox when window goes to collapsed state
